GoAgent配置

翻墙 goagent

goagent一直是大家非常喜爱的翻墙工具,网上的配置方法很多,比如说这个,图文并茂,但我配置的过程中还是遇到了一些问题,现把这些问题记录一下供大家参考

  • goagent下载地址https://github.com/goagent/goagent

  • 在上传application的时候也就是执行 python uploader.zip 命令的时候出现一个错误: 
    Goagent Attribute Error: can’t set attribute 
    网上的解决办法全是让取消 "两步验证",但是经过我的测试发现仅有这一步还不够,还需要打开 "不够安全的应用的访问权限"最终设置如图

  • 在上传完成并修改了 local\proxy.ini 文件之后,即可运行 
    sudo ./goagent-gtk.py 弹出客户端 这时发现了里面有一个warning要求安装gevent,虽然只是一个warning,但是没有这个东西goagent无法正常工作,所以在一阵折腾之后找到了http://pan.baidu.com/s/1sjQ8gWH这两个可用的python包,把它们分别解压并在文件夹中运行  sudo python setup.py install  安装好之后goagent就可以用了
